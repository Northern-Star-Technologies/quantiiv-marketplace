# Quantiiv Marketplace for Claude Code

Official Quantiiv plugins for Claude Code.

## Installation

### 1. Add the marketplace

```bash
claude marketplace add quantiiv-marketplace https://github.com/Northern-Star-Technologies/quantiiv-marketplace.git
```

### 2. Install the plugin

```bash
claude plugin install quantiiv@quantiiv-marketplace
```

Or browse available plugins:

```bash
claude plugin discover
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| **quantiiv** | Query Quantiiv analytics data — sales, products, pricing, elasticity — directly from Claude Code |

## Updating

When plugins are updated in this marketplace, users get the latest version automatically on their next Claude Code session.
